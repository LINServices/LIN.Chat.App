# LIN Allo App

LIN Allo para Windows y Android es una bifurcación del proyecto LIN Allo Web.

Cliente nativo construido en .NET Maui & Blazor Hybrid App.


## Características

- Multiplataforma (Android (APK) y Windows)
- Mensajes
- Grupos de personas
- Integrado con Emma Assistant
